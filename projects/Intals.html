  • Developed a C library to handle arithimetic operation of arbitrary length non-negative integers represented as null-terminated string of ASCII characters
  • Implemented operations include
    • Addition
    • Compare
    • Subtraction / Absolute difference
    • Multiplication
    • Modulo / Remainder
    • Power
    • GCD / HCF
    • Fibonacci Number
    • Factorial
    • Binomial Coefficient
    • Maximum Element
    • Minimum Element
    • Search in Array
    • Binary Search in Sorted Array
    • Sort an Array
    • Coin Row Problem
  • Designed the solution to minimize heap allocation and have zero memory leaks
